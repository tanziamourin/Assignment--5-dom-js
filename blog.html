Question-1: What are the different ways to select an element in the DOM ?

There are 5 main ways to select elements in JavaScript:
1. Selects an element by its (id).
2. Selects elements by their class name.
3. Selects elements by their tag name.
4. Selects the first element that matches a CSS selector.
5. Selects all elements that match a CSS selector (returns a NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML - Changes the HTML and text inside an element.
innerText - Get only visible text (ignores hidden text).
textContent - Gets all the text, even hidden ones, but no HTML tags.

Question-3: What is event delegation in the DOM ?

Event delegation means instead of adding an event listener to each individual element (like buttons), add just one listener to a parent element. When an event happens, it "bubbles up" from the child to the parent, so the parent can handle it.

Question-4: What is event bubbling in the DOM ?

Event bubbling is when an event starts at the target element (where the event happened) and then "bubbles up" to its parent elements, eventually reaching the the document.

Question-5: How do you create, add, and remove elements using JavaScript ?

For HTML element create - document.createElement().
For add the created element to the webpage - appendChild().
For remove an element - remove() or removeChild().
